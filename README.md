# ICope - AI-Powered Blood Analysis Platform

A modern web platform showcasing ICope's innovative AI-powered blood analysis research project. The website presents the research, methodology, and impact of using artificial intelligence for automated blood sample analysis.

## Features

- Modern, responsive design
- Interactive statistics and animations
- Comprehensive project information
- Mobile-friendly interface

## Structure

- `index.html` - Main webpage structure
- `styles.css` - Styling and layout
- `script.js` - Interactive features and animations
- `images/` - Project images and assets

## Setup

1. Clone this repository
2. Ensure all images are placed in the `images/` directory
3. Open `index.html` in a modern web browser

## Required Images

Please add the following images to the `images/` directory:
- microscope.jpg
- ai-analysis.jpg
- mobile-app.jpg

- HTML5
- CSS3
